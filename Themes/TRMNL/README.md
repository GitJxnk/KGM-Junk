# Kogama Embedded Terminal

A lightweight userscript that adds a customizable terminal overlay and dark theme to **kogama.com**. Inspired by Claude AI color schemes

## Features

* Username detection
* Utility commands (open games, profiles, reload, etc.)
* Full-site dark theme inspired by Claude AI
* Clean monospace styling across interface elements

## Commands

* `help` — list commands
* `cls` — clear terminal
* `echo <text>` — print text
* `reload` — refresh the page
* `game <id>` — open a game
* `profile <id>` — open a profile
* `mylevel` — open your levels
* `leaderboard` — open the leaderboard

## Notes

* Some CSS overrides may affect newer UI elements
* Username and terminal settings are stored locally
